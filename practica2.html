Bienvenido a mi pagina HTML


HTML es el lenguaje con el que se escriben las paginas web. Las paginas web pueden ser vistas por el usuario mediante un tipo de apicacion llamada navegador.

Este lenguaje nos permite aglutinar textos, sonidos e imagenes y combinarlos a nuestro gusto.

El HTML nos permite la introduccion de referencias a otras paginas por medio de los enlaces hipertexto.
